## Reporting a Vulnerability

I take the security of Scalpel Racer seriously. If you believe you have found a security vulnerability in Scalpel Racer, please report it to me as described below.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please send an email to `xkilldash9x@proton.me`.

You should expect to receive a response within 24 hours. If for some reason you do not, please follow up with me to ensure I received your original message.

I prefer all communications to be encrypted.
